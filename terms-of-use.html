Article 2 — Objet de l'Application

SocialVault est une application Android proposant les fonctionnalités suivantes :

  • Téléchargement de vidéos publiées sur Facebook, X (anciennement Twitter), Instagram, TikTok et Pinterest ;
  • Téléchargement d'images publiées sur TikTok et Instagram ;
  • Sauvegarde des statuts WhatsApp (stories) dans le stockage local de l'appareil ;
  • Conversion de fichiers vidéo stockés localement en fichiers audio (MP3) ;
  • Stockage sécurisé de fichiers personnels dans un Coffre-fort local protégé par code PIN ou mot de passe.

L'ensemble des médias téléchargés, convertis ou protégés dans le Coffre-fort est stocké exclusivement dans la mémoire locale de l'appareil de l'utilisateur.
